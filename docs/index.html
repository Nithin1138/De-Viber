De-Viber

A local-first CLI that scans Lovable and Bolt exported projects for vendor lock-in dependencies, platform-specific config coupling, and critical security gaps.

npx deviber-cli analyse ./your-project --offline

Trust & Transparency

🔒 Offline & Private

De-Viber runs 100% locally on your machine. It never uploads your code, configuration, or environment keys to any external servers.

⚠️ Heuristic Scan Only

This is a pattern-matching scanner. It cannot guarantee 100% portability or complete security. Always review highlighted areas manually.

🚫 No Automated DB Migrations

Exiting an AI platform requires database migrations. De-Viber identifies lock-in, but database contents and auth tables must be migrated manually.

🛠️ Open Source & Community Driven

We believe code independence belongs to the developers. All rules are open for inspection, contributions, and improvements.
